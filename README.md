# KsgFrame
该库主要包含快速开发框架与个人学习的资料
         
##### 0、KT
##### 1、MVP
##### 2、MVVM
##### 3、RxJava+Retrofit （网络请求框架）
##### 4、StaggeredList（瀑布流+Ac转场动画）
##### 5、Viewpager+Player+Image（包含视频与图片和自适应高度的VP）
##### 6、TouchGrid（手势拖拽Recyclerview）
##### 7、GestureView（签名或手势识别）
##### 8、LoadPage（页面Loading）
##### 9、ListPlayer（列表播放器）
##### 10、Player（播放器框架）
##### 11、鱼儿游动小动画（动画练习）
